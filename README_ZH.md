# Batch File Renamer

批量文件重命名工具，支持多种重命名模式和预览功能。

## 功能特性

- 多种重命名模式
  - 添加前缀/后缀
  - 文本替换（支持正则表达式）
  - 序号命名
  - 大小写转换
  - 日期时间命名
  
- 文件管理
  - 递归处理子目录
  - 文件类型过滤
  - 实时预览
  - 撤销功能
  
- 友好的图形界面
  - 直观的操作界面
  - 拖拽支持
  - 批量操作
  - 操作历史记录

## 安装


```bash
# 克隆项目
git clone https://github.com/yourusername/Batch-File-Renamer.git
cd Batch-File-Renamer

# 运行程序
python -m renamer.gui
```



## 使用说明

1. 启动程序后，点击"选择目录"选择要处理的文件夹
2. 在左侧文件列表中选择要重命名的文件
3. 在右侧选择重命名模式并设置参数
4. 点击"预览"查看重命名效果
5. 确认无误后点击"执行重命名"

## 重命名模式说明

### 添加前缀/后缀
- 在文件名开头或结尾添加固定文本
- 示例：`photo.jpg` -> `vacation_photo.jpg`

### 文本替换
- 替换文件名中的指定文本
- 支持正则表达式匹配
- 示例：`IMG_001.jpg` -> `Photo_001.jpg`

### 序号命名
- 按序号重新命名文件
- 可自定义起始数字和位数
- 示例：`file1.txt`, `file2.txt` -> `001.txt`, `002.txt`

### 大小写转换
- 转换为大写、小写或首字母大写
- 示例：`MyFile.TXT` -> `myfile.txt`

### 日期时间
- 使用文件的修改时间或创建时间命名
- 可自定义日期格式
- 示例：`document.pdf` -> `20260131_143052.pdf`

## 系统要求

- Python 3.7+

## 注意事项

- 重命名前建议先预览确认
- 程序会自动避免文件名冲突
- 可使用撤销功能恢复操作
- 建议在重要文件操作前先备份
